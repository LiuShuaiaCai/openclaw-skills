# 小红书学术日报内容模板

## 图片模板设计建议

### 封面图（3:4 竖版，1080x1440 推荐）

**风格要求**：
- 学术风格，简洁现代
- 主色调：#2B5CFF（学术蓝）或 #7B68EE（紫罗兰）
- 背景：白色或浅灰 #F5F5F5
- 字体：思源黑体/苹方（正文），圆润字体（标题）

**内容布局**：
```
┌─────────────────────────┐
│                         │
│   📚 学术日报           │
│                         │
│   2026年4月18日         │
│                         │
│   ┌─────────────────┐   │
│   │  今日新文 1,234  │   │
│   │       篇        │   │
│   └─────────────────┘   │
│                         │
│   [学科分布图示]        │
│                         │
│   数据来源：CrossRef    │
└─────────────────────────┘
```

### 内页图（可选）

1. **学科分布饼图**
2. **Top 机构排行榜**
3. **热门期刊列表**

## 文案模板

### 标题（20字以内）

```
📚 今日学术速报 | X月X日学术圈发生了什么
🔬 今日论文速递 | X篇新研究来袭
📖 学术日报 | 今天学术界都在研究什么？
💡 今日学术热点 | X篇新论文发布
```

## 发布建议

1. **最佳发布时间**：
   - 工作日：7:30-8:30（通勤）
   - 午休：12:00-13:00
   - 晚间：20:00-22:00

2. **配图建议**：
   - 封面图使用统计数据可视化
   - 内页可添加感兴趣的论文截图
   - 建议添加 3-9 张图片

3. **互动引导**：
   - 评论区可引导"需要某领域论文推荐"
   - 点赞收藏高的内容可再发续集
